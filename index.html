Using Full-Stack Web Development

Wanderlust is a full-stack web application inspired by Airbnb, allowing users to explore, create, and manage property listings. Built using Node.js, Express, MongoDB Atlas and EJS, it follows a robust MVC architecture that keeps the codebase organized and maintainable.

The platform features secure user authentication and authorization, enabling registered users to add listings, write reviews, and manage their own content. Authorization ensures that only the owner of a listing can edit or delete it, protecting user data and access.

Cloudinary is integrated for seamless image uploads, while Multer handles file processing. The interactive map functionality is powered by Mapbox, allowing users to visually explore all listed properties with location-specific data. The UI is fully responsive and styled using Bootstrap 5, offering a smooth and intuitive browsing experience.

Using Full-Stack Web Development

Question Bank Portal is a collaborative full-stack web application designed for managing and accessing college-level question papers. Built using Node.js, Express, MongoDB Atlas, and EJS, the project follows a clean MVC architecture, ensuring organized and scalable code structure.

The platform supports secure authentication and role-based authorization. Teachers and students have separate access controls — teachers can upload, edit, and delete question papers, while students can view and search them based on branch, semester, and subject. Teachers can also assign tasks or homework, adding an extra layer of interaction.

This project was developed as a team collaboration, using Git for version control and Render for live deployment. It demonstrates real-world skills in backend development, user access control, cloud integration, and academic tool design.
